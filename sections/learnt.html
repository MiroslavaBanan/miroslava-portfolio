Aroken School is a hands-on program in modern frontend development built around real-world tasks and professional standards. The course starts with a solid review of HTML, CSS, and JavaScript fundamentals, enriched with practical techniques used by working developers. As the program progresses, it dives deeper into professional layout, semantics, responsive interfaces, and advanced UI patterns, followed by modern tools such as React, Node.js, and working with APIs.

A dedicated module covers WordPress development: installing and configuring sites, creating custom themes and plugins, working with PHP, and adapting functionality to real client needs. Beyond technical skills, the course focuses on soft skills: communication, client acquisition, negotiation, requirement analysis, and building professional relationships. Community is a key part of the experience, with collaborative projects and peer support.

Regular seminars and webinars with industry developers provide insight into real workflows, challenges, and career paths.

This website was created for a small private transportation service operating routes between Poland and Kaliningrad. The goal was to design a simple one-page site that stands out among competitors and clearly presents key information without unnecessary navigation.

I developed the website within approximately one week using WordPress, with additional HTML and CSS for interactive elements and custom layout details. The site was adapted for all device types and included basic performance optimization and SEO setup. The focus was on clear structure, typography, and visual clarity.

Through this project, I gained experience working with small businesses, experimenting with design solutions, and delivering a complete product within a very short timeframe.
